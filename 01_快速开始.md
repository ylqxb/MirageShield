# 快速开始

## 环境准备

### 系统要求
- **操作系统**：Windows 10+（推荐 Windows 11，仅在此环境测试通过）或 Linux (未测试)
- **Python**：3.8~3.11（推荐 3.9 或 3.10，3.12+ 可能存在 asyncio 兼容性问题）
- **硬件**：推荐 4 核 CPU，8GB 内存，50GB 存储空间（最低配置可降低，但可能影响性能）
- **网络**：稳定的网络连接，支持 IPv4/IPv6
- **Docker**：用于部署蜜罐（可选但推荐，不使用 Docker 会导致蜜罐功能受限）

### 软件安装

**Windows 系统**：
1. 安装 Python 3.8+：从 [Python 官网](https://www.python.org/downloads/) 下载并安装
2. 安装 Git：从 [Git 官网](https://git-scm.com/downloads) 下载并安装
3. 安装 Docker Desktop：从 [Docker 官网](https://www.docker.com/products/docker-desktop) 下载并安装
4. 配置环境变量：确保 Python 和 Git 都已添加到系统 PATH

**Linux 系统**：
```bash
# 安装 Python 3.8+
sudo apt update
sudo apt install python3 python3-pip python3-venv

# 安装 Git
sudo apt install git

# 安装 Docker
sudo apt install docker.io
sudo systemctl start docker
sudo systemctl enable docker
sudo usermod -aG docker $USER
```

## 一键部署

### 方法一：使用一键部署脚本

**Windows 系统**：
1. 下载 `install_windows.bat` 脚本
2. 以管理员身份运行脚本
3. 脚本会自动：
   - 克隆代码库
   - 创建虚拟环境
   - 安装依赖
   - 配置环境变量
   - 启动系统

**Linux 系统**：
1. 下载 `install_linux.sh` 脚本
2. 赋予执行权限：`chmod +x install_linux.sh`
3. 运行脚本：`./install_linux.sh`
4. 脚本会自动完成所有部署步骤

### 方法二：手动部署

1. **克隆代码库**
```bash
git clone https://github.com/yourusername/MirageShield.git
cd MirageShield
```

2. **创建并激活虚拟环境**
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux
python3 -m venv venv
source venv/bin/activate
```

3. **安装依赖**
```bash
pip install -r requirements.txt
```

4. **配置环境变量**
创建 `.env` 文件：
```dotenv
# 管理员密码
ADMIN_PASSWORD=your_secure_password

# 加密密钥
ENCRYPTION_KEY=your_encryption_key

# 水印密钥
WATERMARK_KEY=your_watermark_key
```

5. **启动系统**
```bash
python start_server.py
```

## 常见问题

### 端口被占用
- **问题**：启动时提示 5000 端口被占用
- **解决**：检查并关闭占用 5000 端口的进程，或修改 `start_server.py` 中的端口配置

### 依赖安装失败
- **问题**：pip 安装依赖时失败
- **解决**：确保网络连接正常，尝试使用国内镜像源：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`

### 智能体启动失败
- **问题**：智能体状态显示为 "inactive"
- **解决**：检查配置文件是否正确，确保环境变量已设置
