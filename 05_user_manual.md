# 用户手册

> **说明**：本文档描述系统设计功能，部分功能可能正在开发中，具体以实际界面为准。

## 系统登录与权限管理

### 1. 系统登录

**登录流程**：
1. 打开浏览器，访问 http://localhost:5000
2. 输入管理员用户名和密码
3. 点击 "登录" 按钮
4. 登录成功后，进入系统主界面

**默认登录信息**：
- 用户名：admin
- 密码：系统首次启动时生成的密码或通过环境变量设置的密码

**密码重置**：
- 如果忘记密码，可以通过环境变量 `ADMIN_PASSWORD` 设置新密码
- 重启系统后，使用新密码登录

### 2. 权限管理

**用户角色**：
- **管理员**：拥有所有系统权限
- **普通用户**：只能查看系统状态，不能修改配置

**权限设置**：
1. 进入 "系统设置" → "用户管理"
2. 点击 "添加用户" 按钮
3. 填写用户信息和角色
4. 点击 "保存" 按钮

**用户管理**：
- 查看用户列表
- 修改用户信息
- 删除用户
- 重置用户密码

## 系统主界面

### 1. 系统状态

**功能说明**：
- 显示系统运行状态
- 显示智能体状态
- 显示防护级别
- 显示系统资源使用情况

**操作步骤**：
1. 登录系统后，默认进入系统状态页面
2. 查看系统运行状态
3. 点击 "刷新" 按钮更新状态

### 2. 智能体管理

**功能说明**：
- 查看智能体状态
- 启动/停止智能体
- 配置智能体参数

**操作步骤**：
1. 点击左侧菜单 "智能体管理"
2. 查看智能体状态
3. 点击 "启动" 或 "停止" 按钮控制智能体
4. 点击 "配置" 按钮修改智能体参数

### 3. 硬件管理

**功能说明**：
- 查看硬件设备列表
- 监控硬件状态
- 管理硬件保护策略

**操作步骤**：
1. 点击左侧菜单 "硬件管理"
2. 查看硬件设备列表
3. 点击设备名称查看详细信息
4. 点击 "编辑" 按钮修改设备保护策略

### 4. 策略配置

**功能说明**：
- 设置系统防护级别
- 配置防御策略
- 管理黑名单和白名单

**操作步骤**：
1. 点击左侧菜单 "策略配置"
2. 调整防护级别滑块
3. 点击 "保存" 按钮应用设置
4. 配置黑名单和白名单

### 5. 日志管理

**功能说明**：
- 查看系统日志
- 查看智能体日志
- 搜索和过滤日志
- 导出日志

**操作步骤**：
1. 点击左侧菜单 "日志管理"
2. 选择日志类型
3. 输入搜索关键词
4. 点击 "搜索" 按钮
5. 点击 "导出" 按钮导出日志

### 6. 系统设置

**功能说明**：
- 配置系统参数
- 管理用户
- 备份和恢复系统
- 更新系统

**操作步骤**：
1. 点击左侧菜单 "系统设置"
2. 选择设置类别
3. 修改设置参数
4. 点击 "保存" 按钮应用设置

## 常见操作场景

### 1. 部署蜜罐检测端口扫描

**场景说明**：
当系统检测到可疑的端口扫描行为时，部署蜜罐来进一步分析攻击者的意图。

**操作步骤**：
1. 进入 "蜜罐管理" 页面
2. 点击 "部署蜜罐" 按钮
3. 选择 "端口扫描蜜罐" 类型
4. 配置蜜罐参数（如开放端口、响应行为等）
5. 点击 "部署" 按钮
6. 查看蜜罐活动日志，分析攻击者行为

### 2. 配置 AI 智能体分析威胁日志

**场景说明**：
配置 Watcher 智能体分析系统日志，自动检测和响应威胁。

**操作步骤**：
1. 进入 "智能体管理" 页面
2. 点击 "Watcher 智能体" 的 "配置" 按钮
3. 设置日志分析参数（如分析频率、威胁阈值等）
4. 配置响应策略（如自动阻断、生成告警等）
5. 点击 "保存" 按钮应用配置
6. 查看智能体分析结果

### 3. 处理威胁告警

**场景说明**：
当系统检测到威胁时，及时处理告警并采取相应措施。

**操作步骤**：
1. 进入 "系统状态" 页面
2. 查看威胁告警列表
3. 点击 "查看详情" 查看告警详细信息
4. 根据告警类型采取相应措施：
   - 对于端口扫描：部署蜜罐
   - 对于恶意IP：加入黑名单
   - 对于异常行为：调整防护策略
5. 点击 "处理" 按钮标记告警为已处理

### 4. 生成安全报告

**场景说明**：
生成系统安全状态报告，用于安全审计和合规检查。

**操作步骤**：
1. 点击左侧菜单 "日志管理"
2. 点击 "生成报告" 按钮
3. 选择报告类型（如每日报告、每周报告、月度报告）
4. 选择报告内容（如威胁检测、系统状态、网络活动）
5. 点击 "生成" 按钮
6. 查看和导出报告

## 系统维护

### 1. 定期备份

**操作步骤**：
1. 进入 "系统设置" → "备份与恢复"
2. 点击 "手动备份" 按钮
3. 选择备份内容
4. 点击 "开始备份" 按钮
5. 查看备份结果

### 2. 系统更新

**操作步骤**：
1. 进入 "系统设置" → "系统更新"
2. 点击 "检查更新" 按钮
3. 如果有更新，点击 "更新" 按钮
4. 等待更新完成
5. 重启系统

### 3. 日志清理

**操作步骤**：
1. 进入 "系统设置" → "日志管理"
2. 点击 "清理日志" 按钮
3. 选择清理范围（如7天前的日志）
4. 点击 "开始清理" 按钮
5. 查看清理结果

## 故障排除

### 1. 系统无法登录

**问题**：无法登录系统

**解决方案**：
- 检查用户名和密码是否正确
- 检查系统是否正常运行
- 检查网络连接是否正常
- 通过环境变量重置管理员密码

### 2. 智能体状态异常

**问题**：智能体状态显示为 "inactive"

**解决方案**：
- 检查智能体配置是否正确
- 检查系统资源是否充足
- 点击 "启动" 按钮重启智能体
- 查看智能体日志，分析错误原因

### 3. 蜜罐部署失败

**问题**：蜜罐部署失败

**解决方案**：
- 检查 Docker 是否正常运行
- 检查网络连接是否正常
- 检查蜜罐配置是否正确
- 查看系统日志，分析错误原因

### 4. 系统性能下降

**问题**：系统响应缓慢

**解决方案**：
- 检查系统资源使用情况
- 清理系统日志
- 优化系统配置
- 考虑升级硬件

## 快捷键与技巧

### 1. 快捷键

| 快捷键 | 功能 |
|-------|------|
| Ctrl + F | 搜索 |
| Ctrl + R | 刷新页面 |
| Ctrl + S | 保存设置 |
| Alt + 1 | 切换到系统状态页面 |
| Alt + 2 | 切换到智能体管理页面 |
| Alt + 3 | 切换到硬件管理页面 |

### 2. 使用技巧

- **快速查看系统状态**：在任何页面点击顶部导航栏的 "系统状态" 按钮
- **快速搜索**：使用页面顶部的搜索框搜索设备、日志或威胁
- **批量操作**：在列表页面使用复选框选择多个项目进行批量操作
- **导出数据**：在任何列表页面点击 "导出" 按钮导出数据
- **深色模式**：点击页面右上角的 "深色模式" 按钮切换主题
